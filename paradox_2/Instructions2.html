INSTRUCTIONS: FIRST DECISION

You will be offered {% if treatment == 3 %} 3 {% else %} 9 {% endif %} mutual funds.

Mutual funds provide portfolio diversification. For this reason, the return on each mutual fund depends on the performance of the financial assets in which the fund is invested.

The total return on each of the {% if treatment == 3 %} 3 {% else %} 9 {% endif %} mutual funds offered to you depends on the performance of eight underlying assets. Specifically, their return is randomly determined as follows: with a 20% chance, each asset yields a £1 return, and with an 80% chance, each asset yields a £0.25 return. Observe that the return of an underlying asset in the mutual fund does not affect the return of any other underlying asset.

The returns of the underlying assets mutual fund A is invested in are displayed in the second raw (columns 1 to 8) of the table. These could have been generated according to the procedure described above.

In particular, apart from the second asset and the fifth asset, whose realized return is £1, the realized return of the other underlying assets is £0.25. Accordingly, the right-most column of the above table indicates the total return of mutual fund A (£3.50), which is given by the sum of the returns of the underlying assets (0.25*6+£1*2).

As far as mutual fund B is concerned, the realized return of the second, the fourth, and the sixth underlying assets is £0.25, and the realized return of the other assets is £1. Accordingly, the total return of mutual fund B is £5.75 (= £1 × 5 + £0.25 × 3).

Recall that in this activity you will be offered {% if treatment == 3 %} 3 {% else %} 9 {% endif %} mutual funds. Depending on the realizations of the random returns of the underlying assets, the total return of each of the {% if treatment == 3 %} 3 {% else %} 9 {% endif %} mutual funds ranges from £2 (if the return of all underlying assets is £0.25) to £8 (if the return of all underlying assets is £1), with intermediate returns of: £2.75, £3.5, £4.25, £5, £5.75, £6.5, and £7.25.
